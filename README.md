# React Neighborhood Map

## What is this?
It's a web application that shows the Bologna area (Italy) and highlights my favourite locations there.

## Technologies used
* Vanilla JavaScript
* ReactJS
* CSS
* JSX
* Google Maps API
* Four Square API

## Start the project
To start developing:
* download the repository
* `yarn install`
* `yarn start`

To test the development mode, run `yarn build`
